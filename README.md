- **対象**: 主要 AI 企業のモデル/機能発表、AI エージェント/開発ツール、OSS/GitHub トレンド

- **公開**: GitHub Pages (Settings → Pages → Source: `main` / `/ (root)`)

記事は `_posts/YYYY-MM-DD-ai-news.md` として追加されます。
